# Dental Clinic — Agent Notes

Marketing site for Denta Care Dental Clinic (Visakhapatnam).

## Structure

- `src/components/site/` — Header, Footer
- `src/components/ui/` — shadcn/ui components
- `src/routes/` — TanStack Router pages
- `src/lib/` — utilities, services, error handling

## Commands

```bash
npm install
npm run dev
```
