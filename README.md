# Kaana Production

WhatsApp business platform: marketing site, API, inbox, and CRM.

## Apps

- `kaana-platform/` — Marketing website (React + Vite)
- `botiq-whatsapp-server/` — Node.js API, WhatsApp webhooks, SQLite
- `botiq/` — Team inbox dashboard
- `propcrm/` — Lead CRM
